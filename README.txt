Go2 2.0 StarterKit
http://gov2.vlsm.org

Version 0.0.1 - 26 Apr 2015

Author: Cyber GovLabs 

---------------------------

Overview:

Gov 2.0 StarterKit adalah sebuah aplikasi kecil berbasis web yang dirancang untuk dapat menggunakan Platform Government 2.0 Indonesia. Dengan menggunakan aplikasi kecil maka aplikasi Anda yang sudah ada dapat memanfaatkan layanan-layanan yang disediakan oleh Platform Gov 2.0 seperti misalnya Single Account. Dalam starterkit ini disertai dengan contoh aplikasi yang menggunakan Single Account. Baca dokumen MANIFEST.txt untuk lebih detil.

---------------------------

Requirements:

- Min PHP 4.1.0 - tested with 4.2.2
- Internet Domain Name
- Web Hosting
